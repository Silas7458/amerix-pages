🤝 New in v7
The Tandem Team — Multi-Agent Coordination
⟶ One Brain, Four Sets of Hands, One Shared Desktop
The Tandem Team is a real-time multi-agent coordination system running on Silas's desktop. Four AI agents share context via files, archives, and direct communication — Proctor sees the screen and coordinates, Claude Code builds in the terminal, Cowork handles non-coding automation, and Browser Use drives web tasks. Silas says "hey team" and they all activate together. This doesn't exist anywhere else yet.

🔑 Callout Cadence

Proctor MUST proactively check Claude Code's terminal via Windows-MCP throughout every session. Monitor status, relay tasks, read output. Don't wait to be asked.

⚠️ Trade-offs
