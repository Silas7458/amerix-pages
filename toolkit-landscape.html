⟶ Your Board of Directors That Never Sleeps
An n8n-hosted deliberation system running in Docker that replaces all local agent-to-agent communication problems. Instead of Proctor and Brother trying to talk through terminal typing and screen snapshots, they trigger an n8n workflow that runs the conversation for them via Claude API calls. 19 workflows, 14 PostgreSQL tables, 8 authenticated API endpoints.
